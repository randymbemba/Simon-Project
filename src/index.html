Super Simon Game

Règle du jeu
Le principe de ce jeu est simple: vous devez mémoriser l'enchaînement de touches lumineuses et le reproduire. Le but étant de reproduire la plus longue série de couleurs / sons générée aléatoirement par le Simon, c'est un jeu sans fin!
Pour démarrer la partie, appuyez sur le bouton «JOUER»
